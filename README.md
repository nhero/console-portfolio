# Console Portfolio

## Installation and Setup

- run `npm i` or `yarn install` to install node modules

- add stocks to portfolio.json
```
{
  "stocks": {
    "SPY": {
      "quantity": 1
    ,
    "GOOG": {
      "quantity": 1
    }
  }
}

```

## Running the program

- running `node portfolio.js` in the console will show you the current value of your holdings
